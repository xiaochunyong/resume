## 简历

* https://resume.ely.me



前提: 已安装`nodejs` `npm`
- 0、`npm i` 安装依赖
- 1、**开发**：命令行运行 `npm run dev` 开启调试服务器
- 2、编辑 `/src/index.html` 文件实时调试
- 3、**生产**：命令行运行 `npm run build` 进行打包
- 4、浏览器打开 `/index.html` 进行预览

> chrome浏览器 - "打印" - 设置无边距 - 即可导出
